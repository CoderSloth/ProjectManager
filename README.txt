ProjectManager Version 1.0 07/27/2013
==============

DESCRIPTION
-----------
A program to manage and edit multiple txt/rtf documents at once in a single window. Also displays a picture.

GOALS
-----
- Make management of files in a project easier
- Do not let files get lost in multiple windows
- Use few clicks to operate views, seamless transitions
- Intuitive design for a standard or limited experience user
- Do away with the annoyances of multiple windows/toolbars with minimalistic but rich experience
- Make solid use of shortcuts for faster movement within a project
- Make full use of monitor real estate
